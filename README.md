## Calculator Android App (계산기 어플)

**안드로이드 계산기 어플 구현**

<br>

- **구현한 기능 소개**
  - 덧셈, 뺄셈, 곱셈, 나눗셈 연산
  - 괄호 및 소수점 연산
  - 연산 결과 10진수 / 16진수 표기

<br>

- **사용한 언어**
  + Java

<br>

+ **개발 환경**
  + macOS Catalina
  + Android Studio
